## Installation

1. Clone the repository:

 ```bash
   git clone https://github.com/3bdlrhman/ClinicApplication/
```

2. Run SQL Scripts
```
.First SchemaScript
.Then DataScript
```

3. Navigate to the project directory:
```
  cd ClinicApplication
```

4. Install dependencies:
```
dotnet restore
```

5. dotnet run
